#### モード選択画面

![モード選択画面](draw/img/portfolio-form-select.svg)

「---」が記されたボックスは、2つともプルダウンメニューである。<br>左の項目は、プロフィールと実績、お知らせ、管理者設定の4つであり、右の項目はそれぞれ以下の通りに設定する。
| 左の項目     | 右の項目   |
| ------------ | ---------- |
| プロフィール | 編集       |
| 実績         | 投稿、編集 |
| お知らせ     | 投稿       |
| 管理者設定   | 変更       |

#### プレビュー画面

各投稿画面、編集画面にあるプレビューボタンをクリックすることで、新規タブに遷移されるページのこと。<br>記入した情報をもとに、ユーザー向け画面で実際にどう表示されるかを確認する。表示される画面は、ユーザー向け画面で表示されるものと変わらない。

### ヘッダーとフッター

各画面には、共通するヘッダーとフッターを表示させる。内容は以下の通りにする。

| ヘッダー                                  | フッター                             |
| ----------------------------------------- | ------------------------------------ |
| 左側：サイトタイトル<br>右側：SNSのリンク | 各ページへのリンク、<br>コピーライト |
